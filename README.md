# Medici_Merchant
Medici Final Project

각자의 소스코드 및 폴더 이름은 이니셜_폴더(파일)이름 으로 작성한다.
* 유니티 버전은 2021.1.20f1 로 합니다.
* AR Foundation 사용
* ex) CMS_Scripts, CMS_Player.cs
* ex) CJJ_Prefabs, CJJ_Player
* ex) CYH_Music, CYH_Sing.mp3
* ex) SYE_Animation, SYE_Player_ani
